# C64 6510 Assembly Game Engine
==============================

A highly optimized game engine written in 6510 assembly language for the Commodore 64, designed to take advantage of Visual Studio Code and Kick Assembler.

## Features
------------

* Highly optimized code for maximum performance
* Support for 8-bit graphics modes (PAL and NTSC)
* Smooth scrolling and collision detection
* Modular architecture for easy game development
* Integration with Visual Studio Code and Kick Assembler for seamless coding experience

## Getting Started
-------------------

### Prerequisites

* Commodore 64 computer
* Kick Assembler installed
* Visual Studio Code with a Kickass plugin installed

### Clone the Repository

```bash
git clone https://github.com/Quintar/c64_game_engine.git
```

### Build and Run

1. Open the project in Visual Studio Code
2. Edit the game/fame_main.asm source file
3. Compile and run the game.
